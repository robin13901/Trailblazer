<objective>
Build the coverage compute service — the FIRST production writer of the
`coverage_cache` table. It iterates all driven intervals + all cached Kfz way
geometry, attributes each way to its containing admin region at levels
4/6/8/9/10, and upserts `driven_length_m` + `total_length_m` per region. Wire
it to re-run (fire-and-forget) after a trip is confirmed so the cache the
`CoverageInvalidator` just deleted is re-populated. Add a `getAllWithCoverage()`
read so the Wave-2 browser can list every region with coverage.

Purpose: Everything else in Phase 8 (pill, browser, sheet) READS `coverage_cache`.
Until this service runs, all `getByRegionId` reads return null (RESEARCH.md
lines 51-58). This plan makes the cache real.
Output: Compute service + provider, a cache-DAO list method, a recompute hook in
the trips inbox repo, and a unit test proving population + re-population.

Algorithm (RESEARCH.md §Pattern 1, lines 124-145; honor Pitfalls 1/2/3):

```
1. await regionLookup.ensureLoaded();   // MAIN isolate — asset bundle not
                                         // reachable off-isolate (Pitfall 1)
2. bounds = await tripsDao.watchUnionBbox().first;
   if (bounds == null) { await cacheDao.deleteAll(); return Ok(0); }
      // no trips -> empty cache is correct
3. allIntervals = await intervalsDao.getAllIntervals();
   group into byWayId: Map<int, List<Interval>>  (see resolver lines 83-88)
4. ways = await waySource.fetchWaysInBbox(
        minLat/minLon/maxLat/maxLon from bounds, throwOnError: false);
      // cache-first; offline gap -> whatever tiles are cached
5. // Accumulators keyed by region_id (osmId.toString()); NO level prefix
   //   (RESEARCH.md line 491 — OSM ids globally unique across levels).
   driven = <String, double>{};   total = <String, double>{};
   const levels = <int>[4, 6, 8, 9, 10];   // level 2 EXCLUDED (Pitfall 2);
                                            // level 9 INCLUDED (RESEARCH line 263)
   var processed = 0;
   for (final way in ways) {
     final c = _centroid(way.geometry);          // mean lat/lon over points
     final wayLen = _polylineLengthMeters(way.geometry);  // Haversine (resolver 156-167)
     final unionLen = byWayId.containsKey(way.wayId)
         ? drivenLengthMeters(byWayId[way.wayId]!)   // interval_union.dart:72
         : 0.0;
     for (final level in levels) {
       final region = await regionLookup.regionAt(c.lat, c.lon, level);
       if (region == null) continue;
       final id = region.osmId.toString();
       total[id] = (total[id] ?? 0) + wayLen;
       if (unionLen > 0) driven[id] = (driven[id] ?? 0) + unionLen;
     }
     // Pitfall 3: regionAt is sync-fast after load but yield periodically so a
     // large way set never starves the UI thread.
     if (++processed % 250 == 0) await Future<void>.delayed(Duration.zero);
   }
6. // Write every region that has any total length. Wipe stale rows first so a
   // region that dropped to 0 coverage disappears.
   await cacheDao.deleteAll();
   final now = DateTime.now();
   var written = 0;
   for (final id in total.keys) {
     await cacheDao.upsert(
       regionId: id,
       drivenLengthM: driven[id] ?? 0,
       totalLengthM: total[id]!,
       updatedAt: now,
       extractVersion: null,   // Phase 10 wires this
     );
     written++;
   }
   return Ok(written);
```
