  <action>
Add a tap handler to `FocusAreaPill` (from Plan 08-04) that opens
`showRegionDetailSheet` (from Plan 08-05) for the region currently under the map.

- Wrap the `GlassPill` in a `GestureDetector(onTap: () => _openSheet(context, ref), child: ...)`.
- `_openSheet(BuildContext context, WidgetRef ref)` async:
  ```
  final camera = ref.read(liveCameraProvider);            // latest reading
  final lookup = ref.read(adminRegionLookupProvider);
  await lookup.ensureLoaded();
  // Resolve the SAME region the pill is showing: walk fallbackLevelsFrom(zoom).
  final zoom = camera?.zoom ?? 16.0;
  AdminRegion? region;
  for (final level in fallbackLevelsFrom(zoom)) {
    region = await lookup.regionAt(camera?.latitude ?? 0, camera?.longitude ?? 0, level);
    if (region != null) break;
  }
  if (region == null) return;                              // nothing under view
  final row = await ref.read(coverageCacheDaoProvider).getByRegionId(region.osmId.toString());
  final rc = RegionCoverage(
    osmId: region.osmId,
    adminLevel: region.adminLevel,
    name: region.nameDe ?? region.name,
    drivenLengthM: row?.drivenLengthM ?? 0,
    totalLengthM: row?.totalLengthM ?? 0,
  );
  if (!context.mounted) return;
  await showRegionDetailSheet(context, rc);
  ```
- Guard `context.mounted` after each await before using `context`.
- Keep the pill visuals from 08-04 unchanged (two centered lines, GlassPill,
  0-dim guard, `withValues(alpha:)`). Do NOT wrap in Flexible/Expanded.
- The `Semantics` label from 08-04 stays; optionally add `button: true` to it
  now that the pill is tappable.
- Package imports only; `dart:async` if needed.

NOTE: the pill can be at Deutschland (level 2) fallback — that is fine for the
SHEET (unlike the browser, the sheet may show any level including 2, since the
pill itself falls back to Deutschland over water/outside regions).
